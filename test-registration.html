Troubleshooting

If the registration page doesn't load:

  1. Make sure both frontend and backend servers are running
  2. Check that port 8081 is not being used by another application
  3. Verify that the database is accessible (PostgreSQL on port 5432)
  4. Check the browser console for any JavaScript errors
